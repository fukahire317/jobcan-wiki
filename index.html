3. 利用方法（出退勤・休憩）基本操作

出退勤と休憩の打刻方法について説明します

3-1. 出勤・退勤基本操作

出勤時の注意事項：
  • 打刻グループ：「所属の部」を選択
  • 備考：作業場所を入力
退勤時の注意事項：
  • 24時を超えて退勤する場合は「夜間モード」を選択
  • 夜間モード選択で翌1時は25時表記になります

3-2. 出退勤時の打刻し忘れ・間違え時基本操作

打刻し忘れの場合：
  1. 「出勤簿」に移動
  2. 対象日付の「ー」を押下
  3. 「＋」を押下
  4. 「時刻」と「備考（理由）」を入力
  5. 申請完了（承認待ち）
打刻間違えの場合：
  1. 「出勤簿」に移動
  2. 対象日付の時間を押下
  3. 「修正」を押下
  4. 正しい「時刻」と「備考（理由）」を入力
  5. 申請完了（承認待ち）

3-3. 休憩基本操作

重要：
  • 15分以上の休憩を取る場合は打刻が必要
  • 休憩開始時：「打刻」押下で「退室」状態に
  • 休憩終了時：再度「打刻」押下で「入室」状態に
12時～13時の昼休憩：
  • 自動的に休憩時間としてカウント（打刻不要）
推奨休憩時間帯：
  • 17:30～18:00（30分）
  • 22:00～22:30（30分）
  • 翌03:00～03:30（30分）
  • 翌08:00～08:30（30分）
  • ※12:00～13:00は全員必須

3-4. 休憩時の打刻し忘れ・間違い時基本操作

1つだけ打刻し忘れた場合：
  1. 「出勤簿」に移動
  2. 「休憩」の時間を押下
  3. 「＋」を押下
  4. 「時刻」と「備考（理由）」を入力
開始・終了両方忘れた場合：
  • 一度に2つ（開始・終了）を入力
  • 休憩開始時間と終了時間を勤務時間内に挟むことで休憩時間とみなされる

4. 利用方法（工数管理）基本操作

工数の入力方法について説明します

4-1. 実績工数入力基本操作

手順：
  1. 「工数管理」画面に移動
  2. 対象日の「工数実績」を押下
  3. 対象のプロジェクトを選択
  4. 作業場所を備考に記載
  5. 割合または時間数で工数を入力
    • 割合：スライダーまたは％で指定
    • 時間数：「0000」または「00:00」形式で入力
  6. 複数ジョブの場合は「＋」で追加
  7. 「確定」を押下
  8. 総労働時間と工数実績が一致していることを確認
  9. 「保存」を押下

5. 利用方法（各種申請）基本操作

申請の種類：
  1. 新規休暇申請
  2. 新規休日出勤申請
  3. 新規残業申請
  4. 新規早出残業申請

5-1. 新規休暇申請基本操作

休暇の種類：
  • 有給（全休）：残有給から1日（7時間30分）消費
  • 半休（午前/午後）：残有給から0.5日（3時間45分）消費
  • 振休：休日出勤の振替休暇（有給消費なし）
  • 夏季休暇：7月～9月に3日取得可能（有給消費なし）
  • 慶事休暇・弔事休暇：事前に上長に相談
申請の修正・取消：
  • 承認前：自分で修正・取消可能
  • 承認後：上長に問い合わせ

5-2. 新規休日出勤申請基本操作

振替休日ありの場合：
  1. 休日出勤希望日を入力
  2. 取得予定の休暇タイプ「振休」を選択
  3. 休暇日を入力
  4. 出勤理由を入力
  5. 「確認」→「申請」
休日出勤のみの場合：
  1. 休日出勤希望日を入力
  2. 取得予定の休暇タイプ「休日出勤のみ」を選択
  3. 出勤理由を入力
  4. 「確認」→「申請」

5-3. 新規残業申請基本操作

重要：20時を超える場合は必ず事前申請を行う

手順：
  1. 残業予定日を入力
  2. 残業終了時刻を入力
  3. 残業理由を入力
  4. 「確認」→「申請」

5-4. 新規早出残業申請基本操作

対象：9時開始予定以外の場合（7時～8時30分、9時30分～11時の間に作業開始）

手順：
  1. 残業予定日を入力
  2. 残業開始時刻を入力
  3. 残業理由を入力
  4. 「確認」→「申請」

目次使用方法

1. はじめに使用方法

本マニュアルについて：ジョブカンの基本的な使用方法と留意点を記載

2. 1日の流れ：打刻及び工数入力使用方法

例：9時業務開始、途中休憩（17時30分～18時）、19時業務終了

2-1. 通常時使用方法

1日の流れ：
  1. 業務開始時：ログイン後、お知らせと打刻エラーを確認し打刻
  2. 昼食休憩：12時～13時は自動処理（打刻不要）
  3. 途中休憩：12時～13時以外の休憩は打刻が必要
  4. 業務終了：打刻を行い退室状態に
  5. 実績入力：必ず退出打刻後に工数を入力

2-2. 注意点使用方法

半休設定：
  • 作業工数 + 半休工数「0345」（3時間45分固定）
全休設定：
  • 工数「0730」（7時間30分固定）
昼休憩なし：
  • 事前に上長へ報告し休憩時間の修正を依頼
深夜・早朝勤務：
  • 5時を境に前日/当日を分ける
  • 夜間モードには切り替えない

3. 申請処理使用方法

申請の種類：
  1. 新規休暇申請：有給・半休・振休・夏季休暇など
  2. 新規休日出勤申請：休日出勤時
  3. 新規残業申請：20時超または10時間超
  4. 新規早出残業申請：9時開始予定以外

4. 月末処理使用方法

月末最終営業日以降のチェック：
  1. 打刻漏れ・打刻エラーチェック
  2. 勤務表チェック
  3. 工数チェック
  4. 交通費チェック
  5. 完了報告
重要：月初第一営業日までに完了させ上長に報告

4-1. 打刻漏れ・エラーチェック使用方法

チェック方法：
  • トップページで「打刻漏れ」や「打刻エラー」を確認
  • 背景色が赤のデータがエラーデータ
  • エラー内容を確認し対処

4-2. 勤務表チェック使用方法

チェック1：出勤簿を確認し、平日に正しく出勤が付けられているか
チェック2：出勤簿と休暇申請を照らし合わせ、有給が正しくつけられているか

4-3. 工数チェック使用方法

確認観点：
  1. 総労働時間＝工数実績か
  2. 備考に勤務場所が入力されているか
  3. 休暇の工番が入力されているか

4-4. 交通費チェック使用方法

確認方法：
  • 各工数の備考と交通費シートを照らし合わせる
  • 工数管理の勤務場所と交通費シートの情報が一致しているか確認

4-5. 完了報告使用方法

完了手順：
  1. チェック完了（打刻漏れ・エラー、勤務表、工数、交通費）
  2. 交通費シートを所定のフォルダに格納
  3. 上長に完了報告メールを送信
